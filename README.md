# 1.9 GDPS Tools Page

tools Page

tool page

designed for use with [lychee](https://codeberg.org/qimiko/lychee)

please set the following environment variables:

- `GDPS_BASE_URL`: the base URL for the GDPS (only accessed server-side)
- `PUBLIC_TURNSTILE_SITE_KEY`: captcha key. leave out to not have captchas on the site, but make sure lychee has captchas disabled first!!
